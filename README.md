# ⚠️Installation
### - Run ```npm install```

# ⚠️Pushing Code
### - Run ```npm run precommit``` first and fix errors before creating ```Pull Request```

# ⚠️Development
### - Switch to ```develop``` branch and create your own branch origin to ```develop```
### - ```git switch develop```
### - follow this branch formatting ```git checkout -b 'name/date'```
